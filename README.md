## Problem Statement:
AtliQ Mart, an FMCG manufacturer in Gujarat, India, faces service issues leading to non-renewal of key customer contracts. They need to track 'On-time' and 'In-full' delivery metrics to improve service levels before expanding to other cities.

## Task List:
As a data analyst, my responsibilities include:

- Creating the required metrics.
- Designing a dashboard based on stakeholder requirements.
- Providing additional insights beyond the specified metrics and dashboard design.
